README.txt - Dakota-Lua-TestThis sub-directory includes some Lua code and Dakota description files developed for Dakota-Lua integration.For the moment, it is a Dakota executable which should be directly called from command line.Here is the description of included files for Dakota:* dakota_lua.in : Dakota input file* lua_project_driver.sh : the command to be called by Dakota Osmose-related files are described below:* Simple_Stream.lua : a simple Model* Stream_Dakota : extended frontend module which includes a simple Pre & Post computing procedures.How to modify : Please see the related documentation on GitHub.
