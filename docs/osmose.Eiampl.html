Eiampl

This class is responsible to prepare the ET models for the solver. local eiampl = osmose.Eiampl(project)

Class function

.new (project)

Initialize a project for the eiampl solver. It parses the processes and the utilities and their streams. If the project is MER type, it add 2 default utilties.

Exemple:

local project = osmose.Project('LuaJam', 'MER')
project:load(
{cip = "ET.Cip"},
{cm1 = "ET.CookingMixing"},
{cm2 = "ET.CookingMixing", with = 'CM2_inputs.csv'}
)
local eiampl = osmose.Eiampl(project)
