Graph

Create a Gnuplot graph. Just create new instance to create plain Gnuplot graph.

Exemple: osmose.Graph(project)

An output can be specified to created. This to be used for multitimes project.

Exemple to create SVG files : osmose.Graph(project, 'svg')

Class function

.new (project, format)

Draws Composite Curves (CC) and Grand Composites Curves (GCC). You can specify an format output.

Exemples: osmose.Graph(project) osmose.Graph(project, 'svg')

.complete (hotCurve, coldCurve, gccCurve)

This function is used to complete the composite curve with points from the hot curve to the cold curve and poits from the cold curve to the hot curve.

.getCompositeCp (streams,t)

If a temperature is between the Tmin and Tmax of a stream, a cumulative slope is calculated.

.splitHotAndCold (project, periode, time)

The hot streams and cold streams are dispatched in different arrays and are sorted by their minimal temperature. Tmin and Tmax are defined for each stream.

.getCumulateHeatLoad (streams,time)

The cumulative Heat Load of the streams is calculated from the Tmin to the Tmax.

.getColdCC (streams)

Create the composite curve for the cold curve.

.getHotCC (streams)

Create the composite curve for the hot curve.
